# Iris Flower Classification

Comparing three machine learning algorithms to classify iris flower species.

## What this project does
- Loads the classic Iris dataset (150 flowers, 3 species)
- Trains and compares 3 ML models: Decision Tree, KNN, and SVM
- All 3 models achieve 100% accuracy
- Visualizes species separation using scatter plots

## Key findings
- All models achieved 100% accuracy due to clear class separation
- Setosa is completely separable from other species
- Petal measurements are stronger separators than sepal measurements

## Tools used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## How to run
Open the notebook in Google Colab and run all cells in order.
